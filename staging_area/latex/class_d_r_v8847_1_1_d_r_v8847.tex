\doxysubsection{Detailed Description}
A motor driver class for the \mbox{\hyperlink{class_d_r_v8847_1_1_d_r_v8847}{DRV8847}} from TI. 

Objects of this class can be used to configure the \mbox{\hyperlink{class_d_r_v8847_1_1_d_r_v8847}{DRV8847}} motor driver and to create one or more objects of the Motor class which can be used to perform motor control.
