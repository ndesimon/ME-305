Detailed Description

Allows control of individual motors.


This task allows users to control the speed of individual motors by receiving user-inputted duty cycles from the UTask.

Generator function that passes the duty cycles to the motor.

This function passes the duty cycle to the method setduty which then sends the value to the motor.

Parameters
taskNameNames the task so that multiple tasks can be run simultaneously.
periodPasses in the period at which the code is run.
kFlagA shared parameter that indicates if the 'k' key has been pressed.
CLCA shared parameter that indicates the value of th duty cycle.
encdataA tuple containing time, position, and delta of the encoder.
wFlagA shared parameter that indicates if the closed loop control keys have been pressed.
