!

Creates the user interface.

Creates the many functionalities required. Motor can be controlled and data can be collected. User is prompted as to what inputs are valid.

Parameters
taskNameNames the task so that multiple tasks can be run simultaneously.
periodPasses in the period at which the code is run.
zFlagA shared parameter that indicates if the 'z' key has been pressed.
cFlagA shared parameter that indicates if the 'c' key has been pressed.
eFlagA shared parameter that indicates if the 'e' key has been pressed.
encDataA tuple containing time, position, and delta of the encoder.
duty1A shared parameter that passes in the value for the duty cycle of motor 1.
duty2A shared parameter that passes in the value for the duty cycle of motor 2.
KpshareA shared parameter that passes in the value for the gain of Kp.
KishareA shared parameter that passes in the value for the gain of Ki.
yshareA shared parameter that passes in the value for the set point of the closed loop controller.
wFlagA shared parameter that indicates if the closed loop control keys have been pressed.
