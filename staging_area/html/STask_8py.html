Detailed Description

!

Calls functions of the DRV8847 module.

This task enables all assigned motors when the user inputs a 'c' or an 's' into the UTask. This task also disables all assigned motors if a fault condition is triggered.

See source code: https://bitbucket.org/bbartl01/mechatronics-labs/src/master/Lab3/STask.py
