\doxysubsection{Detailed Description}
A sensor driver class for the \mbox{\hyperlink{class_b_n_o055_1_1_b_n_o055}{BNO055}}. 

Objects of this class can be used to configure the \mbox{\hyperlink{class_b_n_o055_1_1_b_n_o055}{BNO055}} IMU device and to create one or more objects of the \mbox{\hyperlink{class_b_n_o055_1_1_b_n_o055}{BNO055}} class which can be used to perform platform control.
