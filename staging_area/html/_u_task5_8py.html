Creates the user interface.

Creates the many functionalities required. Motor can be controlled and data can be collected. User is prompted as to what inputs are valid.

Parameters
taskNameNames the task so that multiple tasks can be run simultaneously.
periodPasses in the period at which the code is run.
eulAngA tuple containing Euler angles head, pitch, and roll.
gyrVelA tuple containing angular velocity about X,Y, and Z.
duty1A shared parameter that passes in the value for the duty cycle of motor 1.
duty2A shared parameter that passes in the value for the duty cycle of motor 2.
KpshareA shared parameter that passes in the value for the gain of Kp.
KdShareA shared parameter that indicates the gain value Kd for the inner loop.
KishareA shared parameter that passes in the value for the gain of Ki.
KpOutA shared parameter that indicates the gain value Kp for the outer loop.
KdOutA shared parameter that indicates the gain value Kd for the outer loop.
KiOutA shared parameter that indicates the gain value Ki for the outer loop.
wFlagA shared parameter that indicates if the closed loop control keys have been pressed.
dFlagA shared parameter that indicates if the value for duty cycle of motor 1 is wanted.
DFlagA shared parameter that indicates if the value for duty cycle of motor 2 is wanted.
abfshareA shared parameter that indicates the position and velocities of the ball for filtering.
yShareA shared parameter that passes in the value set motor angle 1.
YSHAREA shared parameter that passes in the value set motor angle 2.
yFlagA shared parameter that indicates if the set motor angle control keys have been pressed.
