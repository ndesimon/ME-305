Detailed Description

Determines the measurements from the IMU.

This task obtains the measurements from the IMU to calculate the calibration coefficients, euler angles and angular velocity. Fills these values into buffers to be calculated in the BNO055 driver class.

See source code: https://bitbucket.org/bbartl01/mechatronics-labs/src/master/Lab4/CTask.py

BNO function that passes the calibration, euler angles, and angular velocities.

This function passes the values of calibration coeffiecients, euler angles, and angular velocities to the method of set_cal_coeff.

Parameters

periodPasses in the period at which the code is run.
