\doxysubsection{Detailed Description}
! 

A motor class for one channel of the DRV8847.

Objects of this class can be used to apply PWM to a given DC motor.

! 

Constantly updates the counter that tracks position.

The position is tracked with a 16-\/bit counter. To prevent the counter from overflowing, a change, or delta between previous counts is computed. If the magnitude of the delta is larger than half of the maximum count (indicating that the counter reset), the delta is corrected such that the reset isn\textquotesingle{}t seen. Adding up all the deltas gives the current position. \mbox{\Hypertarget{classclosedloop_1_1ClosedLoop_a9977f9a73888261059c38e25671d0cbc}\label{classclosedloop_1_1ClosedLoop_a9977f9a73888261059c38e25671d0cbc}}
