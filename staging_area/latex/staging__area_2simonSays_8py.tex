\doxysubsection{Detailed Description}
Displays Morse patterns on Nucleo LED and asks user to repeat them. 

Implements a FSM, shown below, to create an interactive game for a user. The game entails blinking of a Nucleo LED in morse code patterns, whereupon the user is asked to repeat said pattern using the Nucleo\textquotesingle{}s blue button.
