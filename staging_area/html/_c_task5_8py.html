Detailed Description

Determines actuation for motor control.

This task computes the actuation for motors. It receives desired velocity values from the UTask, measured velocity values from the ETask, and sends the required duty cycle to the MTask.

See source code: https://bitbucket.org/bbartl01/mechatronics-labs/src/master/Lab4/CTask.py

ClosedLoop function that passes the values for closed loop control.

This function passes the euler angles, angular velocities, gain values, and flags to read and write the values used in closed loop control.

Parameters
taskNameNames the task so that multiple tasks can be run simultaneously.
periodPasses in the period at which the code is run.
CLC_OUTA shared parameter that indicates the outer loop reference angle limits.
CLC_INA shared parameter that indicates the inner loop saturation limits.
eulAngA tuple containing the angular position for x,y,z.
wFlagA shared parameter that indicates if the closed loop control keys have been pressed.
KpShareA shared parameter that indicates the gain value Kp for the inner loop.
KdShareA shared parameter that indicates the gain value Kd for the inner loop.
KiShareA shared parameter that indicates the gain value Ki for the inner loop.
KpOutA shared parameter that indicates the gain value Kp for the outer loop.
KdOutA shared parameter that indicates the gain value Kd for the outer loop.
KiOutA shared parameter that indicates the gain value Ki for the outer loop.
dutyA shared parameter that indicates the value of the duty cycle for the motor.
gyrVelA tuple the angular velocities for x,y,z.
numEulAn input parameter for angles to determine if we are in the x or y axis for platform.
numGyrAn input parameter for angular velocities to determine if we are in the x or y axis for the platform.
dFlagA shared parameter that indicates if the value for duty cycle of motor 1 are wanted.
DFlagA shared parameter that indicates if the value for duty cycle of motor 2 are wanted.
abfshareA shared parameter that indicates the position and velocities of the ball for filtering.
posIdxAn input parameter for the position to determine if we are in the x or y axis for the ball.
velIdxAn input parameter for the velocities to determine if we are in the x or y axis for the ball.
RefValA shared parameter that indicates the value of the reference velocity.
yFlagA shared parameter that indicates if the set motor angle control keys have been pressed.
