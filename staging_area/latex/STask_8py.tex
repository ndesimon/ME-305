\doxysubsection{Detailed Description}
! 

Calls functions of the DRV8847 module.

This task enables all assigned motors when the user inputs a \textquotesingle{}c\textquotesingle{} or an \textquotesingle{}s\textquotesingle{} into the UTask. This task also disables all assigned motors if a fault condition is triggered.
